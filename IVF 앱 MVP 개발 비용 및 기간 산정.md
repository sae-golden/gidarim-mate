### MVP 총 비용 산정

#### 시나리오 1: 최소 MVP (단일 플랫폼)
- 범위: iOS 또는 Android 중 하나
- 핵심 기능만 포함
- **예상 비용: $40,000-$60,000 (약 5,200만-7,800만원)**
- **개발 기간: 3-4개월**

#### 시나리오 2: 표준 MVP (크로스 플랫폼)
- 범위: iOS + Android (Flutter 사용)
- 모든 핵심 기능 포함
- **예상 비용: $70,000-$100,000 (약 9,100만-1.3억원)**
- **개발 기간: 4-6개월**

#### 시나리오 3: 풀 MVP (확장 기능 포함)
- 범위: iOS + Android + 웹 관리자 페이지
- 처방전 OCR, 프로토콜 템플릿 포함
- **예상 비용: $120,000-$180,000 (약 1.56억-2.34억원)**
- **개발 기간: 6-9개월**

## 비용 최적화 전략

1. **Flutter 사용**: 단일 코드베이스로 iOS/Android 동시 개발 (30-40% 비용 절감)
2. **MVP 우선 접근**: 핵심 기능만 먼저 개발
3. **기존 라이브러리 활용**: 캘린더, 알림 등 검증된 라이브러리 사용
4. **클라우드 서비스**: Firebase, AWS 등 관리형 서비스 활용
5. **단계적 개발**: Phase 1 → Phase 2 → Phase 3 순차 개발

## 추천 MVP 개발 계획

### Phase 1: Core MVP (3개월, 약 6,000만원)
- 기본 약물 스케줄 등록
- 주사 부위 마킹 (기본)
- 푸시 알림
- 사용자 인증

### Phase 2: 확장 (2개월, 약 3,000만원)
- 카카오톡 연동
- 주사 부위 로테이션 추천
- 프로토콜 템플릿

### Phase 3: 고도화 (2개월, 약 4,000만원)
- 처방전 OCR
- 데이터 분석/리포트
- 병원 연동 준비

**총 예상 비용: 약 1.3억원 (7개월)**
