## 6. 주요 UX 포인트

### 1️⃣ 한눈에 보기
- 메인 화면에서 현재 단계와 전체 흐름을 동시에 표시
- 사용자가 "나 지금 어디쯤이지?" 한눈에 파악 가능

### 2️⃣ 오늘의 할 일 명확하게
- 오늘 해야 할 주사/약을 위에 크게 표시
- 완료/미완료 상태 한눈에 보기
- 과거 완료한 것도 보여주기

### 3️⃣ 주사 부위 로테이션 (핵심)
- 완료 버튼 누르면 바로 위치 입력 팝업
- 어제 맞은 곳 표시 + 내일 추천 위치 표시
- 자동으로 로테이션 추천 알고리즘 작동

### 4️⃣ 캘린더로 전체 흐름 파악
- 월간/주간 캘린더로 완료율 시각화
- 어느 날 빠뜨렸는지 한눈에 보기

### 5️⃣ 초기 설정 최소화
- 병원 선택 → 단계 선택 → 일정 수정
- 3단계만으로 기본 설정 완료
