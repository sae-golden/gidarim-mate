# 시험관메이트 - 앱 아이콘 디자인 요청

## 앱 정보
- **앱 이름**: 시험관메이트
- **앱 성격**: 시험관 시술 중인 예비 엄마들을 위한 약물/치료 기록 앱
- **앱 톤앤매너**: 따뜻함, 희망, 동반자, 응원

## 앱 컬러
- Primary: 보라색 (#9B7ED9, #7C3AED)
- 서브: 연보라, 라벤더, 핑크

## 아이콘 컨셉 제안

### 컨셉 1: 두 손이 감싸는 하트 (추천 ⭐)
- 두 손이 하트를 소중히 감싸는 형태
- "메이트 = 동반자" 느낌
- 따뜻하고 신뢰감 있는 느낌
- 보라색 그라데이션 배경

### 컨셉 2: 하트 안의 아기 발자국
- 보라색 하트 안에 작은 아기 발자국
- 아기를 기다리는 마음 표현
- 희망적이고 사랑스러운 느낌

### 컨셉 3: 새싹 + 하트
- 작은 새싹이 자라는 모습
- 옆에 작은 하트
- 생명의 시작, 성장, 희망
- 연보라/핑크 배경

### 컨셉 4: 연결된 두 개의 하트
- 큰 하트와 작은 하트가 연결된 모습
- 엄마와 아기의 연결
- 부드러운 곡선

### 컨셉 5: 달과 별
- 초승달과 작은 별
- 기다림, 희망, 꿈
- 포근한 밤하늘 느낌
- 진한 보라 배경에 밝은 달/별

## 디자인 가이드
- **스타일**: 미니멀하고 부드러운 일러스트
- **형태**: 둥근 모서리 정사각형 (iOS/Android 앱 아이콘)
- **색감**: 보라색 그라데이션 기반, 따뜻한 느낌
- **피해야 할 것**: 
  - 너무 복잡한 디테일
  - 차가운 느낌
  - 의료/병원 느낌 (주사기, 약 등 직접적 표현 X)
  - 슬픈 느낌
