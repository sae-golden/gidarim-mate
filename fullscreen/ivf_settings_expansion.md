# 시험관메이트 - 설정 확장 기획

## 1. 병원 정보 (미래 확장용)

### MVP 단계
```
┌─────────────────────────────────────────┐
│ 🏥 병원 정보                            │
├─────────────────────────────────────────┤
│ 병원명         [차병원 강남            ]│
│ 담당 선생님    [김OO 교수님            ]│
│ 전화번호       [02-1234-5678       📞 ]│  ← 탭하면 바로 전화
│ 메모           [3층 생식의학센터       ]│
└─────────────────────────────────────────┘
```

### 미래 확장 (빅데이터 활용)

**1) 병원별 프로토콜 템플릿**
```
병원 선택: [차병원 강남 ▼]

💡 이 병원에서 자주 사용하는 프로토콜이에요
┌─────────────────────────────────────────┐
│ 📋 길항제 프로토콜 (인기 1위)           │
│    고나엘에프, 오비드렐, 세트로타이드   │
│    평균 10-12일                    [선택]│
├─────────────────────────────────────────┤
│ 📋 장기 프로토콜                        │
│    루프론, 퓨레곤, 프로게스테론         │
│    평균 14-16일                    [선택]│
└─────────────────────────────────────────┘
```

**2) 병원별 통계 (익명화)**
```
📊 차병원 강남 통계 (2024)
├── 평균 채취 난자: 12.3개
├── 평균 과배란 기간: 11.2일
├── 평균 채취→이식: 5.1일
└── 🔒 개인정보 비식별화 처리됨
```

**3) 수집 가능한 빅데이터**
| 데이터 | 활용 |
|--------|------|
| 병원별 프로토콜 | 템플릿 자동 추천 |
| 약물 조합 패턴 | 신규 사용자 가이드 |
| 채취 일정 패턴 | 예상 일정 추천 |
| 성공률 통계 | 병원 선택 참고 (조심스럽게) |
| 지역별 분포 | 난임 지원 정책 연계 |

**4) B2B 확장 가능성**
- 병원에게 익명화된 트렌드 리포트 제공
- 제약사에게 약물 사용 패턴 데이터 (동의 기반)
- 난임 연구 기관에 연구 데이터 제공

## 2. 알림 설정 확장

### 현재 (기본)
```
알림 받기         [ON]
미리 알림         [ON]
미리 알림 시간    [10분 전 ▼]
```

### 확장안

```
┌─────────────────────────────────────────┐
│ 🔔 알림 설정                            │
├─────────────────────────────────────────┤
│                                         │
│ 📍 투약 알림                            │
│ ─────────────────────────────────────── │
│ 알림 받기                          [ON] │
│ 미리 알림                          [ON] │
│ 미리 알림 시간              [10분 전 ▼] │
│ 미완료 시 재알림                   [ON] │
│ 재알림 간격                  [5분 후 ▼] │
│ 방해금지 모드에서도 알림           [ON] │
│                                         │
│ 👨 파트너 알림                          │
│ ─────────────────────────────────────── │
│ 파트너에게 알림 보내기             [ON] │
│ 카카오톡 연동                   [연동됨] │
│ 내가 미체크 시 파트너에게 알림     [ON] │
│                                         │
│ 📅 일정 알림                            │
│ ─────────────────────────────────────── │
│ 병원 방문 전날 알림                [ON] │

│                                         │

│ ─────────────────────────────────────── │
│ 앱 업데이트 소식                   [ON] │
│ 난임 지원 정책 안내                [ON] │
│ 유용한 정보 & 팁                   [ON] │
│ 이벤트 및 혜택                     [ON] │
│                                         │
└─────────────────────────────────────────┘
```

### 알림 종류별 상세

#### 1) 투약 알림 (핵심)
| 설정 | 설명 | 기본값 |
|------|------|--------|
| 알림 받기 | 복용 시간에 알림 | ON |
| 미리 알림 | 복용 전 미리 알림 | ON |
| 미리 알림 시간 | 5분/10분/15분/30분 전 | 10분 전 |
| 미완료 시 재알림 | 체크 안 하면 다시 알림 | ON |
| 재알림 간격 | 5분/10분/15분 후 | 5분 후 |
| 방해금지 무시 | 중요 주사는 무조건 알림 | OFF |

#### 2) 파트너 알림
| 설정 | 설명 | 기본값 |
|------|------|--------|
| 파트너 알림 | 남편에게도 알림 | OFF |
| 카카오톡 연동 | 카톡으로 알림 전송 | - |
| 미체크 시 알림 | 내가 안 하면 남편에게 | OFF |

#### 3) 일정 알림
| 설정 | 설명 | 기본값 |
|------|------|--------|
| 병원 방문 전날 | 내일 병원 방문 알림 | ON |
| D-Day 알림 | 채취일/이식일 당일 응원 | ON |
| 판정일 알림 | 판정일 응원 메시지 | ON |

#### 4) 마케팅/정보 알림
| 설정 | 설명 | 기본값 | 수익화 |
|------|------|--------|--------|
| 앱 업데이트 | 새 기능 안내 | ON | - |
| 난임 지원 정책 | 정부/지자체 지원금 안내 | ON | - |
| 유용한 정보 | 영양제, 생활 팁 등 | ON | 콘텐츠 제휴 |
| 이벤트 및 혜택 | 할인, 프로모션 등 | OFF | **광고 수익** |

## 3. 수익화 연계 알림

### 광고/제휴 가능 영역

```
💜 시험관메이트 추천

┌─────────────────────────────────────────┐
│ 🧴 [AD] 난임 영양제 특가                │
│    엽산+비타민D+코엔자임Q10             │
│    이번 주 30% 할인                     │
│                              [자세히 >] │
└─────────────────────────────────────────┘

┌─────────────────────────────────────────┐
│ 🏥 [정보] 서울시 난임 지원금 확대       │
│    최대 110만원 지원                    │
│    신청 방법 안내                       │
│                              [확인하기] │
└─────────────────────────────────────────┘

┌─────────────────────────────────────────┐
│ 📚 [콘텐츠] 이식 후 생활 가이드         │
│    전문의가 알려주는 주의사항           │
│                                [읽기 >] │
└─────────────────────────────────────────┘
```

### 제휴 가능 업체
| 카테고리 | 예시 | 수익 모델 |
|----------|------|-----------|
| 영양제 | 엽산, 비타민D, 오메가3 | CPA/CPS |
| 한의원 | 난임 한방 치료 | 소개비 |
| 보험 | 난임 특약 보험 | 리드 |
| 병원 | 난임 전문 병원 | 광고/소개비 |
| 출산용품 | 임신 성공 시 | CPS |
| 심리상담 | 난임 전문 상담 | 소개비 |

### 알림 발송 시점 (똑똑하게)
| 시점 | 알림 내용 |
|------|-----------|
| 첫 시작 | 난임 지원금 안내 |
| 과배란 중 | 영양제 추천 |
| 이식 전 | 착상 돕는 생활 팁 |
| 판정 대기 | 심리 케어 콘텐츠 |
| 성공 시 | 임신 초기 가이드, 출산용품 |
| 실패 시 | 심리상담 안내, 응원 콘텐츠 (광고 X) |
