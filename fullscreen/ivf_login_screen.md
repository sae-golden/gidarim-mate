# 시험관메이트 - 로그인 화면 수정

## 개요
로그인 화면을 더 따뜻하고 감성적인 느낌으로 수정

---

## 수정 사항

| 항목 | 변경 전 | 변경 후 |
|------|---------|---------|
| 앱 이름 | IVF 약물 알림 | **시험관메이트** |
| 아이콘 | 💉 주사기 | **🌱 새싹** |
| 글귀 | 다시 만나서 반가워요! | **희망을 향한 오늘 하루** |
| 소셜 로그인 | 카카오 + Google | **카카오만** (Google 삭제) |

---

## 코드 수정 가이드

### 1. 앱 이름 변경
```dart
Text(
  '시험관메이트',
  style: TextStyle(
    fontSize: 24,
    fontWeight: FontWeight.bold,
  ),
)
```

### 2. 글귀 변경
```dart
Text(
  '희망을 향한 오늘 하루',
  style: TextStyle(
    fontSize: 16,
    color: Colors.grey[600],
  ),
)
```

### 3. 아이콘 변경
```dart
// 기존 주사기 아이콘 대신 새싹 이미지 또는 아이콘 사용
Container(
  width: 80,
  height: 80,
  decoration: BoxDecoration(
    color: Color(0xFFE9D5FF),  // 연보라 배경
    borderRadius: BorderRadius.circular(20),
  ),
  child: Center(
    child: Text('🌱', style: TextStyle(fontSize: 40)),
    // 또는 Image.asset('assets/icons/sprout.png')
  ),
)
```

### 4. Google 로그인 버튼 삭제

## 아이콘 옵션

새싹 아이콘을 직접 만들거나, 아래 방법 사용:

| 방법 | 설명 |
|------|------|
| 이모지 | `🌱` 텍스트로 표시 |
| 커스텀 이미지 | 디자이너가 만든 새싹 아이콘 |
| 아이콘 패키지 | `flutter_icons` 등에서 새싹 아이콘 |
