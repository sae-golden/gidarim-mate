# IVF 앱 - 홈 화면 개선 요청

## 개요
홈 화면을 심플하게 정리하고, 예비 엄마들의 마음을 따뜻하게 안아주는 감성적인 어투로 변경합니다.

## 현재 문제점
- 화면이 너무 복잡함 (현재 단계, 진행 상황, 단계별 흐름 등 중복 요소)
- "오늘의 할 일" 같은 딱딱한 투두리스트 느낌
- 새 약물 추가 버튼이 홈 중간에 묻혀있음
- 치료기록 탭과 중복되는 정보가 많음

## 삭제할 요소
| 요소 | 이유 |
|------|------|
| 현재 단계 카드 | 치료기록 탭에서 이미 보여줌 |
| 새 약물 추가 버튼 | 하단 탭 가운데로 이동 |
| 진행 상황 (7%, 원형 그래프) | 치료기록 탭과 중복 |
| 단계별 흐름 | 치료기록 탭과 중복 |

## 유지/추가할 요소
| 요소 | 처리 |
|------|------|
| 오늘의 한마디 | ✅ 유지 |
| 오늘의 할 일 → "오늘도 한 걸음" | ✅ 유지 + 어투 변경 + 최상단으로 |
| 다가오는 일정 → "곧 만나요" | ✅ 추가 |

## 하단 네비게이션 (FAB 스타일)

**새 약물 추가 버튼을 하단 탭 가운데에 FAB(Floating Action Button)으로 배치**

```
┌──────┬──────┬────────┬──────┬──────┐
│  🏠  │  📅  │   ➕   │  📊  │  ⚙️  │
│  홈  │캘린더│        │치료  │ 설정 │
│      │      │        │기록  │      │
└──────┴──────┴────────┴──────┴──────┘
                  ↑
         보라색 원형 버튼
         다른 탭보다 위로 튀어나옴
         탭하면 약물 추가 화면으로 이동
```

### FAB 버튼 스타일
- 크기: 56-60px 원형
- 색상: Primary 보라색 (#9B7ED9)
- 아이콘: + (흰색)
- 위치: 하단 탭 바에서 위로 살짝 튀어나옴 (offset -20px 정도)
- 그림자: elevation 효과

## 어투 가이드 (따뜻한 감성)

예비 엄마들의 마음을 따뜻하게 안아주는 어투를 사용합니다.

### 변경 전 → 변경 후

| 기존 (딱딱함) | 개선 (따뜻함) |
|---------------|---------------|
| 오늘의 할 일 | 🌸 오늘도 한 걸음 |
| 1/3 완료 | 2/3 💪 |
| 완료 버튼 | 했어요 ✓ |
| 다가오는 일정 | 🌼 곧 만나요 |
| 안녕하세요 | 안녕하세요 👋 |

### 인사말 예시
- "오늘 하루도 응원해요"
- "오늘도 함께해요"
- "좋은 아침이에요"

### 오늘의 한마디 예시 (랜덤 표시)
- "조급해하지 않아도 괜찮아요. 천천히, 당신의 속도로."
- "푹 쉬어야 내일도 힘낼 수 있어요"
- "오늘 하루도 수고했어요 💜"
- "잘하고 있어요. 정말로."
- "당신은 이미 충분히 잘하고 있어요"
- "하루하루가 기적을 향한 걸음이에요"

## 각 섹션 상세

### 1. 헤더
```dart
// 시간대별 인사말
String getGreeting() {
  final hour = DateTime.now().hour;
  if (hour < 12) return "좋은 아침이에요 ☀️";
  if (hour < 18) return "오늘 하루도 응원해요";
  return "오늘도 수고했어요 🌙";
}
```
- 알림 아이콘 (우측 상단)

### 2. 오늘의 한마디
- 💜 아이콘 + 제목
- 랜덤으로 따뜻한 문구 표시
- 카드 배경: 연한 보라색 또는 흰색

### 3. 오늘도 한 걸음 (약물 리스트)
- 🌸 아이콘 + 제목 + 진행률 (예: 2/3 💪)
- 오늘 복용할 약물 리스트
- 완료된 항목: 체크 표시 + 흐리게
- 미완료 항목: 빈 원 + [했어요] 버튼
- 항목 탭 시 완료 처리

### 4. 곧 만나요 (다가오는 일정)
- 🌼 아이콘 + 제목
- 가장 가까운 중요 일정 1-2개만 표시
- 예: "채취일까지 D-5", "이식일까지 D-10"
- 일정 없으면 섹션 숨김

## 데이터 모델

```dart
class HomeData {
  String greeting;           // 시간대별 인사말
  String dailyMessage;       // 오늘의 한마디
  List<TodayMedication> medications;  // 오늘 약물 리스트
  List<UpcomingEvent> upcomingEvents; // 다가오는 일정
}

class TodayMedication {
  String name;
  String time;          // "오전 07:00"
  String dosage;        // "100mg"
  MedicationType type;  // pill, injection, etc.
  bool isCompleted;
}

class UpcomingEvent {
  String title;         // "채취일"
  int dDay;             // -5 (D-5)
  DateTime date;
}
```

## 우선순위

### Phase 1 (필수)
1. 불필요한 섹션 삭제 (현재 단계, 진행 상황, 단계별 흐름)
2. 하단 탭에 FAB 버튼 추가 (약물 추가)
3. "오늘의 할 일" → "오늘도 한 걸음" 어투 변경

### Phase 2
4. "곧 만나요" 섹션 추가
5. 시간대별 인사말 적용
6. 오늘의 한마디 랜덤 문구
