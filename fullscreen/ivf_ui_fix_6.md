# IVF 앱 - UI/UX 수정 요청 6가지

## 1. 약물 추가 > 캘린더 기본값 변경

### 현재 문제
- 기본으로 14일간 선택되어 있음
- 동그라미가 색이 채워진 상태

### 수정 요청
- **기본값: 아무것도 선택 안 됨**
- 동그라미 스타일: 채워진 원 → **테두리만 있는 원**

```
Before:
  ●  ●  ●  ●  ●  (채워진 원, 14일 선택됨)

After:
  ○  ○  ○  ○  ○  (빈 원, 선택 안 됨)
  
선택하면:
  ○  ○  ●  ●  ●  (선택한 날짜만 채워짐)
```

### 스타일
```dart
// 선택 안 됨
Container(
  decoration: BoxDecoration(
    shape: BoxShape.circle,
    border: Border.all(color: primaryColor, width: 2),
    color: Colors.transparent,  // 빈 원
  ),
)

// 선택됨
Container(
  decoration: BoxDecoration(
    shape: BoxShape.circle,
    color: primaryColor,  // 채워진 원
  ),
)
```

## 2. 홈 > "오늘도 수고했어요" 뱃지 색상 변경

### 현재 문제
- 민트색 톤이라 잘 안 보임

### 수정 요청
- 더 눈에 띄는 색상으로 변경
- 추천: 연보라 / 코랄핑크 / 따뜻한 오렌지

```
Before:
  [오늘도 수고했어요 💜]  ← 민트색 (안 보임)

After:
  [오늘도 수고했어요 💜]  ← 연보라 또는 코랄핑크
```

### 색상 제안
```dart
// 옵션 1: 연보라 (앱 톤과 맞음)
Color badgeColor = Color(0xFFE8DEF8);  // 연보라 배경
Color textColor = Color(0xFF7C4DFF);   // 보라 텍스트

// 옵션 2: 코랄핑크 (따뜻한 느낌)
Color badgeColor = Color(0xFFFFE4E1);  // 연분홍 배경
Color textColor = Color(0xFFE91E63);   // 핑크 텍스트

// 옵션 3: 따뜻한 오렌지
Color badgeColor = Color(0xFFFFF3E0);  // 연주황 배경
Color textColor = Color(0xFFFF9800);   // 오렌지 텍스트
```

## 3. 하단바 > + 버튼 아래 텍스트 추가

### 현재 문제
- + 버튼만 있고 텍스트 없음
- 다른 탭은 아이콘 + 텍스트 있음

### 수정 요청
- + 버튼 아래에 텍스트 추가

```
Before:
  🏠    📅         ➕         📊    ⚙️
  홈   캘린더              치료기록  설정

After:
  🏠    📅         ➕         📊    ⚙️
  홈   캘린더     약물추가    기록   설정
```

## 4. 하단바 > "치료기록" 명칭 변경

### 현재 문제
- "치료"라는 단어가 시험관 고객에게 맞지 않음
- 치료 = 아프다는 느낌

### 수정 요청
명칭 변경 제안:

| 현재 | 제안 1 | 제안 2 | 제안 3 |
|------|--------|--------|--------|
| 치료기록 | 기록 | 나의 여정 | 시술기록 |

**추천: "기록"** (심플하고 부담 없음)

```
Before:
  🏠    📅         ➕         📊    ⚙️
  홈   캘린더              치료기록  설정

After:
  🏠    📅         ➕         📊    ⚙️
  홈   캘린더     약물추가    기록   설정
```

## 5. 캘린더 > "이번 달 통계" 제거

### 현재 문제
- 이번 달 통계 (완료율, 연속 완료, 총 완료) 표시됨
- 불필요한 정보

### 수정 요청
- **이번 달 통계 섹션 전체 삭제**

```
Before:
┌─────────────────────────────────────────┐
│ 📊 이번 달 통계                         │
│   90%      9일       30회              │
│  완료율   연속완료   총 완료            │
└─────────────────────────────────────────┘

After:
(삭제)
```

## 6. 캘린더 > 날짜별 약물 섹션 수정

### 현재 문제
- "12월 27일 약물" - 어감이 딱딱함
- 캘린더 아래에 위치함

### 수정 요청
1. **어감 변경**: "약물" → 더 따뜻한 표현
2. **위치 변경**: 캘린더 위로 이동

### 제안
| 현재 | 제안 1 | 제안 2 | 제안 3 |
|------|--------|--------|--------|
| 12월 27일 약물 | 12월 27일 | 오늘의 복용 | 12.27 오늘 |

**추천: 날짜만 표시** (예: "12월 27일" 또는 "12.27 금요일")

### 레이아웃 변경
```
Before:
┌─────────────────────────────────────────┐
│ 📅 12월                          < > │
│ [캘린더 그리드]                         │
├─────────────────────────────────────────┤
│ 📋 12월 27일 약물                       │
│ ✓ 아스피린                             │
│ ✗ FSH 주사                             │
└─────────────────────────────────────────┘

After:
┌─────────────────────────────────────────┐
│ 📋 12월 27일 금요일                     │
│ ✓ 아스피린                             │
│ ✗ FSH 주사                             │
├─────────────────────────────────────────┤
│ 📅 12월                          < > │
│ [캘린더 그리드]                         │
└─────────────────────────────────────────┘
```
