## 11. 주사 부위 로테이션 알고리즘 (개선)

```
배 부위를 9개 구역으로 나눔:

    ┌─────────────┐
    │  1  2  3   │
    │  4  5  6   │
    │  7  8  9   │
    └─────────────┘

로테이션 규칙:
1. 사용자가 입력한 위치 기록
2. 최소 3일 간격 유지 (같은 구역 재사용 전)
3. 시간이 지나면서 패턴 학습
4. 사용자의 선호도 반영

예시:
Day 1: 구역 1 → Day 2: 구역 2 → Day 3: 구역 3
Day 4: 구역 4 → Day 5: 구역 5 → Day 6: 구역 6
Day 7: 구역 7 → Day 8: 구역 8 → Day 9: 구역 9
Day 10: 다시 구역 1로 (9일 경과, 안전)
```

## 12. 사용자 여정 (User Journey)

```
1️⃣ 앱 설치 & 회원가입
   ↓
2️⃣ 입력 방식 선택 (사진/음성/텍스트/직접)
   ↓
3️⃣ 약물 일정 자동 인식 & 기간 설정
   ↓
4️⃣ 최종 확인 (총 횟수, 기간 등)
   ↓
5️⃣ 메인 대시보드에서 오늘의 할 일 확인
   ↓
6️⃣ 매일 완료 버튼 누르고 주사 위치 입력
   ↓
7️⃣ 자동 로테이션 추천 받기
   ↓
8️⃣ 캘린더에서 진행 상황 확인
   ↓
9️⃣ 단계 변경 시 자동으로 약물 일정 업데이트
   ↓
🔟 치료 완료 후 리포트 다운로드
```

## 13. 핵심 UX 원칙

1. **최소한의 입력**: 사진/음성/텍스트 중 선택 → 자동 인식
2. **한눈에 파악**: 대시보드에서 현재 상태, 오늘의 할 일, 진행률 모두 보기
3. **자동 계산**: 총 횟수, 기간, 완료율 모두 자동 계산
4. **스마트 알림**: 매일 다른 위치 추천, 미완료 시 재알림
5. **시각화**: 캘린더, 그래프, 진행률 바로 직관적 표현
6. **유연성**: 언제든 약물 일정 수정 가능
